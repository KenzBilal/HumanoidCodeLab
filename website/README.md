# Humanoid Code Lab — Product Website

This is the official landing page and documentation site for **Humanoid Code Lab**, a professional IDE for programming 3D robots with AI.

## Features
- **3D Robot Simulation**: Live Three.js-powered humanoid robot wireframe.
- **Multi-Page Experience**: Dedicated pages for Home, Download, Documentation, Support, and About.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile.
- **Glassmorphism Aesthetic**: Modern, robotic-themed UI with premium animations.

## Tech Stack
- [Vite](https://vitejs.dev/) - Build Tool
- [React](https://reactjs.org/) - UI Framework
- [React Router](https://reactrouter.com/) - Navigation
- [Three.js](https://threejs.org/) - 3D Graphics
- [Lucide React](https://lucide.dev/) - Icons

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Production Build
```bash
npm run build
```
The production bundle will be generated in the `dist/` directory.
