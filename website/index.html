// FEATURES

Everything You Need to
Bring Robots to Life

A complete desktop environment for learning, building, and experimenting with humanoid robot motion.

⌨️

Monaco Editor

The same engine behind VS Code. Full IntelliSense auto-complete for every robot.* command, live syntax highlighting, and bracket matching.

✨

Multi-Provider AI

Describe your robot routine in plain English. Generate scripts via Google Gemini, OpenAI GPT-4o, or Anthropic Claude — switch providers in one click.

🐛

Visual Step Debugger

Execute your script line-by-line. The editor highlights the current instruction while the 3D robot performs the action in real-time.

🎓

Training Curriculum

Interactive coding puzzles that teach robot programming concepts. Write code, click "Check", and the system validates your logic against goal conditions.

🎬

Keyframe Animator

Create custom animations with per-joint rotation sliders and a timeline. Scrub through your creation and play it back on the 3D model instantly.

🔒

Secure by Design

API keys encrypted via the OS native keychain using Electron's safeStorage. Nothing is stored in plain text — ever.

// SCRIPTING DSL

Write Code.
Control the Robot.

Python-like syntax with variables, loops, conditionals, and 25+ built-in commands. No boilerplate — just express your intent.

  • ✓ Variables: steps = 3
  • ✓ Conditionals: if / elif / else
  • ✓ Loops: for i in range(N)
  • ✓ Comparison: == != < > <= >= and or not
  • ✓ Custom animations: robot.play(animation='dance')
robot_script.py
1# Walk and wave routine 2steps = 5 3 4robot.walk.forward(steps=steps) 5robot.right_hand.wave(times=3) 6 7if steps > 3: 8 robot.turn.left(angle=90) 9 robot.jump(height=1.5) 10else: 11 robot.head.nod(times=2) 12 13for i in range(3): 14 robot.walk.forward(steps=i) 15 robot.wait(seconds=0.5) 16 17robot.stand_on_one_leg(leg='left') 18robot.reset()

Help & Support

Got questions? We've got answers. Reach out anytime.

Humanoid Code Lab is a free, open-source desktop application (Electron) that lets you write Python-like scripts to control a 3D humanoid robot in real-time. It features a built-in Monaco code editor, multi-provider AI script generation, a visual step-by-step debugger, and interactive training puzzles.

Yes! Humanoid Code Lab is 100% free and open-source under the Apache 2.0 license. You can download, use, modify, and redistribute it without any cost.

We support Google Gemini (2.0 Flash), OpenAI (GPT-4o), and Anthropic Claude (3.7 Sonnet). You bring your own API key and can switch between providers at any time within the app.

Yes! The app includes a serial port interface that can stream commands to physical robot hardware via USB. This feature is designed for educational robotics kits and Arduino-based humanoid builds.
